# Premier League Analysis

This is an interactive **Premier League Team Analysis** web app built with **Python**, **Streamlit**, **Pandas**, and **Matplotlib**. The app then analyze teams, matches, head-to-head stats, and performance trends with interactive charts and tables.

## Live Demo

Try the app here: [Premier League Analysis](https://premier-league-analysis-mlz28zdegm4fzofe429lg9.streamlit.app/)

## Features

- Team-wise match summary: Wins, Draws, Losses, Goals For/Against, Points.
- Performance Charts: Goals per match, Goals by season, Win/Draw/Loss distribution.
- Discipline Visualization: Yellow and Red cards visualized with colored bars.
- Shots & Accuracy: Total shots and shot accuracy per team.
- Home vs Away Comparison.
- Team Comparison: Compare two teams side by side.
- Head-to-Head Analysis: Match history and summary for selected season or all seasons.
- Download team data as CSV.

## Tech Stack

- Python 3
- Streamlit
- Pandas
- Matplotlib
